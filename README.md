# CyberGuard – Modular Security Toolkit

## Overview

CyberGuard is a cybersecurity-focused platform designed to provide multiple security analysis and monitoring capabilities in a single environment.

The project combines defensive security, threat analysis, vulnerability assessment, integrity monitoring, and security automation concepts to help students and security enthusiasts understand real-world cybersecurity workflows.

---

## Features

* File Integrity Monitoring
* Vulnerability Assessment
* JWT Security Analysis
* CVSS Risk Scoring
* Phishing Detection
* Password Security Analysis
* Threat Intelligence Utilities
* Security Monitoring Components
* Log Analysis Modules
* Cryptographic Operations

---

## Technologies Used

* Python
* Flask
* Docker
* HTML/CSS
* Security APIs
* Linux Environment

---

## Project Goals

The primary goal of CyberGuard is to create a modular security platform that demonstrates practical cybersecurity concepts including:

* Threat Detection
* Risk Assessment
* Security Monitoring
* Secure Coding
* Defensive Security Operations
